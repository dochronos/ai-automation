# 🤖 AI Automation Workflow

This project explores how to automate repetitive business workflows by combining **AI tools** with low-code/no-code platforms such as **n8n** or **Make**.

## 🔹 Project Objectives
- Automate manual business tasks with AI-powered nodes.
- Showcase integrations (e.g., email, spreadsheets, APIs).
- Document the process with clear step-by-step explanation.
- Share learnings on LinkedIn as part of a professional growth series.

## 📂 Repository Structure
- `docs/` → Documentation and diagrams of automation flows.
- `src/` → Optional Python helper scripts.
- `README.md` → Main documentation.

## 🛠️ Tech Stack
- n8n / Make
- OpenAI API
- REST APIs
- Python (for extra helpers)

## 📌 Notes
This repository complements the **Subte Dashboard project**, forming part of a multi-week learning and professional showcase initiative.
